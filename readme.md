#  daycare

## About

Daycare is an application that enables you to manage your daycare.

## Technologies
- Node.js
- ExpressJs
- MongoDb
- Docker compose


## Project setup

### Clone repository
```
$ git clone https://github.com/lillian-busingye/daystar
```

### Install dependencies
```
npm install
```

### Start mongoDB
```
docker-compose up -d
```

### Run your application
```
npm start
```
